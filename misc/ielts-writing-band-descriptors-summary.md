> **Source**: IELTS Official Band Descriptors (Updated May 2023)
> **适用考试**: Academic & General Training
> **评分范围**: Band 0-9
> **核心洞见**: 四项标准各占25%，理解评分逻辑 = 战略性提分

## Why This Matters

理解 IELTS 评分标准不是为了"应试技巧"，而是为了**战略性地分配备考时间**。

> "A script must fully fit the positive features of the descriptor at a particular level. **Bolded text indicates negative features that will limit a rating.**"

**关键洞见**：
1. **四项标准完全平等** — 语法再好，任务完成度不够也拿不到高分
2. **负面特征是硬伤** — 粗体标注的问题会直接限制得分上限
3. **Task 1 和 Task 2 评分标准不同** — 分别是 Task Achievement vs Task Response

## 四大评分标准详解

### 1. Task Achievement / Task Response (TA/TR)

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                    TASK 1 vs TASK 2 的核心区别                              │
├─────────────────────────────────────────────────────────────────────────────┤
│                                                                             │
│   TASK 1 (Task Achievement):                                                │
│   ┌─────────────────────────────────────────────────────────────────────┐   │
│   │   Academic:                                                         │   │
│   │   • Overview (概述) — 必须有！                                       │   │
│   │   • Key features selected (关键特征筛选)                            │   │
│   │   • Data/figures to support (数据支撑)                              │   │
│   │                                                                     │   │
│   │   General Training:                                                 │   │
│   │   • All bullet points covered (所有要点覆盖)                        │   │
│   │   • Clear purpose (目的明确)                                        │   │
│   │   • Appropriate tone (语气恰当)                                     │   │
│   └─────────────────────────────────────────────────────────────────────┘   │
│                                                                             │
│   TASK 2 (Task Response):                                                   │
│   ┌─────────────────────────────────────────────────────────────────────┐   │
│   │   • Position presented (立场呈现)                                   │   │
│   │   • Ideas extended and supported (观点展开和支撑)                   │   │
│   │   • Directly answers the question (直接回答问题)                    │   │
│   └─────────────────────────────────────────────────────────────────────┘   │
│                                                                             │
└─────────────────────────────────────────────────────────────────────────────┘
```

#### Band 对比表：Task Achievement/Response

| Band | Task 1 Academic | Task 1 GT | Task 2 |
|------|-----------------|-----------|--------|
| **9** | 完全满足所有要求 | 完全满足所有要求 | 深入探讨，立场清晰充分展开 |
| **8** | 关键特征**熟练选择**并清晰呈现 | 所有要点清晰呈现并适当拓展 | 充分回应，观点相关且有支撑 |
| **7** | **有清晰概述**，数据合理分类 | 所有要点覆盖，语气一致恰当 | 主要部分回应，有时过度概括 |
| **6** | 概述尝试，数据选择恰当 | 要点覆盖，目的基本清晰 | 立场相关但结论可能不清晰 |
| **5** | 关键特征覆盖不足，可能缺数据支撑 | 要点呈现但覆盖不充分 | 立场表达但展开不清晰 |

#### 致命错误 (会限制评分):

```
Task 1:
❌ 没有 Overview (Academic)
❌ 遗漏 bullet points (GT)
❌ 包含无关、不准确或重复信息
❌ 只罗列数据不分析趋势

Task 2:
❌ 没有明确立场
❌ 偏题或误解题目
❌ 观点不展开或缺乏支撑
❌ 大量重复
```

### 2. Coherence & Cohesion (CC) 连贯与衔接

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                    COHERENCE & COHESION 进阶路径                            │
├─────────────────────────────────────────────────────────────────────────────┤
│                                                                             │
│   Band 5: 有组织但不完全逻辑                                                │
│      │    句子之间连接不流畅                                                │
│      │    衔接词使用有限/过度                                               │
│      ▼                                                                      │
│   Band 6: 基本连贯，有整体进展                                              │
│      │    衔接词使用有效果但可能机械                                        │
│      │    指代和替换缺乏灵活性                                              │
│      ▼                                                                      │
│   Band 7: 逻辑组织，清晰进展                        ← 关键跃升点            │
│      │    衔接手段灵活使用（含指代、替换）                                  │
│      │    段落有效支撑整体连贯                                              │
│      ▼                                                                      │
│   Band 8: 信息逻辑排序，衔接管理良好                                        │
│      │    偶尔有小失误                                                      │
│      │    段落使用充分恰当                                                  │
│      ▼                                                                      │
│   Band 9: 毫不费力地跟随                                                    │
│           衔接极少引起注意                                                  │
│           段落管理熟练                                                      │
│                                                                             │
└─────────────────────────────────────────────────────────────────────────────┘
```

#### CC 评分关键词解码

| Band | 关键词 | 含义 |
|------|--------|------|
| 9 | "effortlessly" | 读者完全不需要努力理解逻辑 |
| 8 | "with ease", "well managed" | 流畅，管理良好 |
| 7 | "clear progression", "flexibly" | 清晰进展，灵活使用 |
| 6 | "generally coherent", "faulty or mechanical" | 基本连贯，但机械或有错误 |
| 5 | "not wholly logical", "not fluently linked" | 不完全逻辑，连接不流畅 |

#### 衔接手段类型

```
┌─────────────────────────────────────────────────────────────────────────────┐
│   COHESIVE DEVICES (衔接手段)                                               │
├─────────────────────────────────────────────────────────────────────────────┤
│                                                                             │
│   1. LINKING WORDS (连接词)                                                 │
│      However, Moreover, Furthermore, In contrast, As a result...            │
│      ⚠️ 过度使用会显得机械 (Band 6 trap)                                    │
│                                                                             │
│   2. REFERENCE (指代)                                                       │
│      this, that, these, those, it, they, such, the former/latter           │
│      ✅ Band 7+ 要求灵活使用                                                │
│                                                                             │
│   3. SUBSTITUTION (替换)                                                    │
│      用 "do so", "one", "the same" 避免重复                                 │
│      ✅ Band 7+ 的关键技能                                                  │
│                                                                             │
│   4. LEXICAL COHESION (词汇衔接)                                            │
│      同义词、上下义词、相关词汇串联                                         │
│      ✅ 高级衔接，不依赖显性连接词                                          │
│                                                                             │
└─────────────────────────────────────────────────────────────────────────────┘
```

### 3. Lexical Resource (LR) 词汇资源

#### Band 对比表：词汇资源

| Band | 范围 | 准确性 | 风格意识 | 拼写 |
|------|------|--------|----------|------|
| **9** | 完全灵活精准 | 非常自然和精妙的控制 | 完全掌握 | 极少错误 |
| **8** | 广泛流畅灵活 | 偶尔用词不当 | 熟练使用不常见/习语 | 偶尔拼写错误，不影响理解 |
| **7** | 足够灵活精准 | 有使用不常见词汇的能力 | 有风格和搭配意识 | 少数错误，不影响清晰度 |
| **6** | 基本足够 | 范围受限或用词不精准 | 风险承担者词汇更广但错误更多 | 有错误但不影响交流 |
| **5** | 有限但基本够用 | 简单词汇使用准确 | 缺乏灵活性，常简化/重复 | 错误可能造成阅读困难 |

#### 词汇进阶策略

```
Band 5 → 6:
├── 避免重复同一个词
├── 尝试使用同义词
└── 注意基本搭配

Band 6 → 7:
├── 使用 less common vocabulary (不常见词汇)
├── 展示 collocation awareness (搭配意识)
├── 控制 word formation errors (构词错误)
└── ⚠️ 不要为炫技而使用不确定的词

Band 7 → 8:
├── Idiomatic expressions (习语表达)
├── Sophisticated control (精妙控制)
├── Precise meanings (精准表意)
└── Natural word choice (自然用词)
```

### 4. Grammatical Range & Accuracy (GRA) 语法多样性与准确性

#### Band 对比表：语法

| Band | 结构多样性 | 准确性 | 标点 |
|------|------------|--------|------|
| **9** | 完全灵活控制 | 极少错误 | 全程恰当 |
| **8** | 广泛结构灵活准确使用 | 多数句子无错误 | 管理良好 |
| **7** | **复杂结构有一定灵活性** | 无错误句子频繁出现 | 基本控制良好 |
| **6** | 简单和复杂句混合，**灵活性有限** | 复杂句准确度不如简单句 | 有错误但少影响交流 |
| **5** | 结构有限且重复 | 复杂句尝试但常有错误 | 可能有问题 |

#### 语法进阶路径

```
┌─────────────────────────────────────────────────────────────────────────────┐
│              GRAMMAR PROGRESSION 语法进阶                                   │
├─────────────────────────────────────────────────────────────────────────────┤
│                                                                             │
│   Band 5: 简单句为主，从句罕见                                              │
│           "Simple sentences predominate"                                    │
│                                                                             │
│   Band 6: 简单句和复杂句混合                                                │
│           但复杂句的准确度不如简单句                                        │
│           "flexibility is limited"                                          │
│                                                                             │
│   Band 7: 复杂结构**有一定灵活性和准确性**       ← 关键跃升                 │
│           无错误句子频繁出现                                                │
│           "variety of complex structures with some flexibility"             │
│                                                                             │
│   Band 8: 广泛结构**灵活准确**使用                                          │
│           多数句子无错误                                                    │
│           "wide range flexibly and accurately used"                         │
│                                                                             │
│   Band 9: 完全灵活控制                                                      │
│           极少错误                                                          │
│           "full flexibility and control"                                    │
│                                                                             │
└─────────────────────────────────────────────────────────────────────────────┘
```

#### 复杂句结构类型

```
必须掌握的结构 (Band 6+):
├── 定语从句 (Relative clauses)
├── 状语从句 (Adverbial clauses)
├── 名词性从句 (Noun clauses)
└── 被动语态 (Passive voice)

进阶结构 (Band 7+):
├── 分词结构 (Participle clauses)
├── 倒装句 (Inversion)
├── 强调句 (Cleft sentences)
├── 虚拟语气 (Subjunctive)
└── 复合复杂句 (Compound-complex sentences)
```

## Band 跃升关键点

### Band 5 → 6: 从"不足"到"基本达标"

| 标准 | Band 5 问题 | Band 6 要求 |
|------|-------------|-------------|
| TA/TR | 关键特征覆盖不足 | 要点都覆盖，有概述尝试 |
| CC | 不完全逻辑，句子连接不流畅 | 基本连贯，有整体进展 |
| LR | 范围有限，常简化重复 | 范围基本足够，意思基本清晰 |
| GRA | 简单句为主，复杂句常出错 | 简单和复杂句混合使用 |

### Band 6 → 7: 从"adequate"到"clear" (关键跃升)

| 标准 | Band 6 → Band 7 关键变化 |
|------|--------------------------|
| TA/TR | "adequately" → "clearly", 有**清晰概述**，数据分类合理 |
| CC | "generally coherent" → "logically organised", **清晰进展** |
| LR | "rather restricted" → "sufficient flexibility", 能用**不常见词汇** |
| GRA | "limited flexibility" → "some flexibility", **复杂结构更准确** |

### Band 7 → 8: 从"clear"到"skilful" (高手进阶)

| 标准 | Band 7 → Band 8 关键变化 |
|------|--------------------------|
| TA/TR | "clearly highlighted" → "**skilfully selected**" |
| CC | "a few lapses" → "occasional lapses", 衔接**管理良好** |
| LR | "some ability" → "**skilful use**" of uncommon/idiomatic items |
| GRA | "some flexibility" → "**flexibly and accurately**", 多数句子无错误 |

## 常见扣分陷阱

### Task Achievement/Response

```
❌ BAND LIMITERS (会限制评分的问题):

Task 1 Academic:
• 没有 Overview (直接限制到 Band 6 以下)
• 只罗列数据不分析趋势
• 遗漏关键数据点
• 包含不准确信息

Task 1 GT:
• 遗漏 bullet points
• 语气不一致 (正式信用非正式语气)
• 目的不清晰

Task 2:
• 没有明确立场
• 偏题或答非所问
• 观点不展开
• 结论与正文矛盾
```

### Coherence & Cohesion

```
❌ COMMON MISTAKES:

Band 6 陷阱:
• 过度使用 "Firstly, Secondly, Finally"
• 机械性使用 "However, Moreover, Furthermore"
• 指代不清 (this/it 指代模糊)

Band 7 陷阱:
• 段落内部逻辑不清
• 缺乏替换和指代的灵活性
• 衔接词使用不准确
```

### Lexical Resource

```
❌ VOCABULARY TRAPS:

• 重复使用同一个词 (尤其是题目中的词)
• 使用不确定的"大词"导致错误
• 搭配错误 (make a decision ✓, do a decision ✗)
• 拼写错误影响理解
```

### Grammar

```
❌ GRAMMAR PITFALLS:

Band 5-6 常见问题:
• 主谓一致错误
• 时态混乱
• 冠词使用错误

Band 6-7 常见问题:
• 复杂句结构不完整
• 关系代词使用错误
• 标点符号问题 (逗号滥用)
```

## Key Takeaways

### For Band 6 Seekers

1. **确保完成所有任务要求** — Task 1 必须有概述，Task 2 必须有立场
2. **使用衔接词但不要过度** — 机械使用是 Band 6 特征
3. **尝试复杂句** — 即使不完全准确，也要展示尝试
4. **注意基本语法** — 主谓一致、时态、冠词

### For Band 7 Seekers

1. **展示灵活性** — 衔接手段、词汇、句式都要多样化
2. **掌握指代和替换** — 这是 Band 7 CC 的关键
3. **使用不常见词汇** — 但要确保准确
4. **复杂句要准确** — 无错误句子要"频繁出现"
5. **清晰的逻辑进展** — 段落内和段落间都要有

### For Band 8+ Seekers

1. **熟练选择和呈现** — "skilfully" 是关键词
2. **灵活准确的语法** — 多数句子无错误
3. **习语和不常见词汇的熟练使用** — 自然而非炫技
4. **衔接"管理良好"** — 几乎不引起注意
5. **极少失误** — 偶尔的小问题不影响整体
