### 4.3 数据层设计原则

| 原则 | 说明 | 实践 |
|------|------|------|
| **单一数据源** | YAML 是唯一的编辑入口 | `rubric-source.yaml`、`boost-source.yaml` |
| **自动同步** | 脚本自动生成派生文件 | `npm run sync` |
| **占位符规范** | _spec 文件只定义格式，不含示例 | `{placeholder}` 格式 |
| **层级化 ID** | ID 包含层级信息，便于追溯 | `DM1-IS02-AT03` |
| **前后端一致** | iOS 和 WebApp 共用同一套数据 | JSON 格式统一 |

### 4.4 API 设计

| 端点 | 方向 | 数据格式 | 说明 |

| `POST /api/mcq/start` | iOS → Server | `{ video_url }` | 开始诊断 |
| Response | Server → iOS | `mcq_questions.json` | 返回 5 道题 |
| `POST /api/mcq/submit` | iOS → Server | `mcq_response.json` | 提交答案 |

### 5.2 产品核心功能

#### MCQ 快速诊断

用户只需回答 5 道选择题，系统就能精准识别动作问题。每道题对应一个错误原子（Atom），题目设计遵循「问题发现导向」原则，让用户在回答过程中产生「Aha Moment」。

#### 错误原子库 (46 个)

涵盖网球正手的所有常见错误，每个错误原子包含：
- 中文名称 + 英文名称
- 错误描述 + 判断标准
- 错误视频示例

#### Boost 训练库 (26 个)

#### 项目 1: 机器人加工系统搭建与在线颤振监测抑制

**背景**: 硕士毕业论文，中德移动交流项目资助

**摘要**: 清华大学 LIMES 实验室机器人加工系统，基于 ABB IRB6700（2.6m 臂展，200kg 负载，末端精度 0.29 mm），集成西门子 PLC、高速电主轴、RealSense 视觉系统、CHOTEST 自动测量探头等，融合 ROS 生态、视觉识别、颤振在线监测与自适应变主轴转速抑制技术。

**核心成果**:
- 系统检测阈值超限后 100 ms 内响应
- 颤振幅值降低超 80%
- 表面粗糙度稳定在 Ra ≈ 2 µm
- 抑振成功率超过 90%

---

#### 项目 2: 铣削自激振动动力学建模与稳定性分析

**摘要**: 面向铣削加工中的自激振动（颤振）动力学建模、稳定性分析与仿真。通过模态分析与力锤激振实验获取主轴–刀具–工件系统的固有频率、阻尼比与频响函数（FRF）。基于再生效应的时滞微分方程（DDE）建立系统模型，结合 MATLAB 实现颤振数值模拟与控制策略验证。

---

#### 项目 3: 镍基高温合金 Inconel718 热力耦合切削仿真

**摘要**: 基于热力学第一定律、胡克弹性及 Johnson–Cook 塑性/损伤本构模型，在 Abaqus 中开展 36 组刀具–工件热–力耦合切削仿真，提取切削力、进给力、切屑厚度及切削区温度场。

---

#### 项目 4: 超声手术刀设计与性能研究

**背景**: 本科毕业论文

**摘要**: 基于纵–扭复合超声振动需求，提出夹心式压电换能器＋斜槽变幅杆方案，采用 Mason 等效电路建模确定 PZT‐8、45钢与 TC4 的关键尺寸以实现 25 kHz 谐振。在 Abaqus 模态及谐响应分析中验证 25±0.2 kHz 谐振和医疗级振幅 100 μm。

---

#### 项目 5: 解魔方机器人设计与实现

**成果**: 2021.8 第四届中国高校智能机器人创意大赛**全国二等奖**（用时 27.717 秒）

**摘要**: 自动化解魔方机器人，采用双臂与电磁手爪结构，结合计算机视觉颜色识别与 Kociemba 两阶段算法实现快速魔方还原，Arduino 控制步进电机与电磁机构。

**技术细节**:
- 机械: 轻质铝型材"门"型双臂框架 + 闭环步进电机
- 视觉: 4×720P 摄像头 + 光罩 + 补光设计
- 电控: PCB 集成 Arduino、MOSFET 驱动与双电源架构

---

#### 项目 6: 加工表面实时重建与颤振量化研究

**摘要**: 提出新的振动能量指标量化铣削颤振与受迫振动对表面质量的影响，通过 1D CNN 基于加速度信号重建加工表面形貌。

**核心成果**:
- Ra 预测误差 < 0.5 µm
- 与 Ra² 相关系数 r = 0.98
- 可集成于数字孪生系统实现实时质量监控
