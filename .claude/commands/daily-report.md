生成今日工作日报：

## 任务

分析今天的对话历史和 P0 任务记录，生成一份面向老板汇报的工作日报。

## 约束

1. 只包含与今日 P0 任务相关的内容
2. 微信友好格式：不用表格、不用 emoji/icon
3. 用"一、二、三"编号
4. 结果导向，不写技术细节
5. 突出业务价值

## 输出格式

```
【今日工作日报 YYYY-MM-DD】

一、[核心交付/实现功能]

核心成果：[一句话总结]

• [具体成果 1]
• [具体成果 2]
• [具体成果 3]

业务价值：[对产品/用户的价值]

———

二、[第二项重要工作]

• [改动点 1]
• [改动点 2]
• [改动点 3]

[补充说明或预期效果]

———

三、[第三项工作/问题解决]

• [问题 1] → [解决方案]
• [问题 2] → [解决方案]

当前状态：[一句话总结当前进度]
```

## 写作原则

1. 业务视角：写用户/产品价值，不写代码细节
2. 结果导向：写完成了什么，不写怎么做的
3. 简洁明了：每条控制在 1-2 行
4. 数字说话：能量化就量化（X 小时、X 轮迭代）
